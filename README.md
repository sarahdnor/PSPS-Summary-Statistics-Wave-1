# PSPS Education Summary Statistics – Wave 1

This repository contains the code, data files, and outputs used to produce summary statistics and analyses for the Philippines Socioeconomic Panel Survey (PSPS) Wave 1 education module(s).

## Repository Contents

### Data/
Contains datasets used throughout the analysis, including survey data and supporting files.

### Stata/
Contains Stata do files for prelininary analysis

### 0_Sandbox.R
Workspace for testing code, exploring variables, and developing analyses before integrating them into the main scripts.

### 1_Individual_Survey.R
Main analysis script for cleaning individual-level survey data, creating variables, and generating summary statistics.

### Summary_Statistics.qmd
Quarto source document used to generate the final summary statistics report.

### Summary_Statistics.html
Rendered HTML version of the summary statistics report.

### W_Random.R
Additional analysis, exploratory work, and supplementary code used during the project.
